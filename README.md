# BUNGURU WA Web Remake

## 繁體中文

2001 年 Windows 免費遊戲《ぶんぐるWA》的非商業網頁復刻版。遊戲圖像、音樂與關卡資料均內嵌於單一 HTML，可直接在現代瀏覽器遊玩。

### 線上遊玩

<https://vinceli-coder.github.io/BUNGURU_WA_web_remake/>

### 操作方式

- `←`／`→`：左右移動
- `Space`：跳躍
- `Esc`：返回標題畫面
- 手機：選單與姓名輸入使用四向搖桿；關卡中用搖桿左右移動、跳躍鍵跳躍

## English

A non-commercial web remake of *Bunguru WA*, a 2001 Windows freeware game. Its artwork, music, and level data are embedded in one HTML file that runs directly in modern browsers.

### Play online

<https://vinceli-coder.github.io/BUNGURU_WA_web_remake/>

### Controls

- `←` / `→`: Move left or right
- `Space`: Jump
- `Esc`: Return to the title screen
- Mobile: The stick is four-way in menus and name entry; during gameplay it moves left/right, while the separate jump button jumps

This page contains no analytics, advertising, tracking, external network requests, publisher personal data, or credentials. Language and rankings are stored only in the visitor's browser via `localStorage`.
